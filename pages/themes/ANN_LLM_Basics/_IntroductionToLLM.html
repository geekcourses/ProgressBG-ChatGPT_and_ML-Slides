How Neural Networks Work

How Neural Networks Work

The Concept

Biological Inspiration

Inspired by the human brain's network of neurons.
Neuron (Node): Receives input, processes it, and passes it on.
Synapse (Weight): Determines the strength of the signal between neurons.

Layers of a Neural Network

Input Layer: Receives raw data (e.g., pixels of an image).
Hidden Layers: Where the "magic" happens – feature extraction and pattern recognition.
Output Layer: Produces the final prediction (e.g., "Cat" or "Dog").

The Learning Process

Forward Propagation: Data flows from input to output.
Loss Function: Measures valid predictions vs actual targets.
Backpropagation: Error is sent back through the network to adjust weights and improve accuracy.

How LLMs Learn

How LLMs Learn

The Transformer Architecture

The Breakthrough: Attention Mechanism

Allows the model to weigh the importance of different words in a sentence, regardless of distance.
Enables understanding of context and relationships (e.g., "bank" of a river vs "bank" for money).

Training Stages

1. Pre-training: Learning from massive datasets (Internet text). The goal: Predict the next token.
Result: A "Base Model" that knows grammar and facts but lacks direction.
2. Fine-tuning (Instruction Tuning): Training on specific Q&A pairs to follow instructions.
3. RLHF (Reinforcement Learning from Human Feedback): Refining the model to be helpful, harmless, and honest.

The Code


from openai import OpenAI
import os
from dotenv import load_dotenv

# Load environment variables
load_dotenv()

client = OpenAI(
    api_key=os.environ.get("OPENAI_API_KEY"),
)

response = client.chat.completions.create(
    model="gpt-4o",
    messages=[
        {"role": "system", "content": "You are a helpful assistant."},
        {"role": "user", "content": "Explain quantum computing in one sentence."}
    ]
)

print(response.choices[0].message.content)
