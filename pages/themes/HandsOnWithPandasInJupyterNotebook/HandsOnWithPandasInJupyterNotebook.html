Working locally with Jupyter Lab

Jupyter Lab Interface

The Launcher: Your starting point to create Notebooks, Consoles, or Terminals.
File Browser: (Left sidebar) Manage files and folders.
The Notebook Area: Where you write code and text.
Code Cells: Execute Python code.
Markdown Cells: Write formatted text (like this slide!).

Running Code

To run a cell:
Click the Play button in the toolbar.
Or press Shift + Enter (Runs and moves to next cell).
Or press Ctrl + Enter (Runs and stays in current cell).

Google Colab Basics

Google Colab: Accessing and Creating Notebooks

Step 1: Go to colab.research.google.com.
Step 2: Log in with your Google Account.
Step 3: Click on New Notebook in the bottom right of the welcome modal, or go to File > New notebook.
Step 4: Rename your notebook by clicking on the file name at the top left.

The Interface & Runtime

Connect: Click "Connect" (top right) to allocate server resources (RAM/Disk).
Code & Text: Use "+ Code" or "+ Text" buttons to add cells.
Files: Click the folder icon on the left sidebar to view temporary files.

Mounting Google Drive

To access files from your Drive:

                from google.colab import drive
                drive.mount('/content/drive')
This allows you to read/write datasets directly to your cloud storage.

Sharing and Exporting

Sharing: Click the Share button (top right) to let others view or edit (just like Google Docs).
Exporting: Go to File > Download.
Download .ipynb to run in Jupyter Lab/Notebook locally.

Introduction to Pandas

What is Pandas?

Overview

Pandas is a fast, powerful, flexible and easy to use open source data analysis and manipulation tool.
It is built on top of the Python programming language.
Key structures:
Series: 1D labeled array (like a column).
DataFrame: 2D labeled data structure (like a table).

Getting Started

Importing Pandas

To use Pandas, you must first import it.
The standard alias is pd.

                import pandas as pd

The DataFrame

Creating a DataFrame

You can create a DataFrame from a dictionary.

                data = {
                    'Name': ['Alice', 'Bob', 'Charlie'],
                    'Age': [25, 30, 35],
                    'City': ['New York', 'London', 'Paris']
                }
                df = pd.DataFrame(data)
                print(df)

Loading Data

Reading CSV Files

Pandas makes it easy to read data from CSV files.

                # Read a CSV file into a DataFrame
                df = pd.read_csv('data.csv')
Other formats supported: Excel, JSON, SQL, etc.

Inspecting Data

Basic Inspection Methods

df.head(): View the first 5 rows.
df.tail(): View the last 5 rows.
df.info(): Summary of the DataFrame (columns, types, non-null counts).
df.describe(): Statistical summary of numerical columns.

Selecting Data

Selecting Columns

Select a single column:

                ages = df['Age']
Select multiple columns:

                name_city = df[['Name', 'City']]

Basic Filtering

Filtering Rows

Select rows based on a condition.

                # Select people older than 28
                older_than_28 = df[df['Age'] > 28]
